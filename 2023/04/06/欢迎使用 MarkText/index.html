📕 Markdown 简介

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的 HTML 页面。 —— 维基百科

正如您在阅读的这份文档，它使用简单的符号标识不同的标题，将某些文字标记为粗体或者斜体，创建一个链接或一个脚注 demo。下面列举了几个高级功能，更多语法请按 Ctrl + / 查看帮助。

复选框 ☑️

使用 - [ ] 和 - [x] 语法可以创建复选框，实现 todo-list 等功能。例如：

  • [x] 已完成事项
  • [ ] 待办事项1
  • [ ] 待办事项2

📗 笔记相关

笔记本和标签 🔖

@(笔记本)[标签A|标签B]语法, 以选择笔记本和添加标签。 绑定账号后， 输入(自动会出现笔记本列表，请从中选择。

笔记标题 📰

自动使用文档内出现的第一个标题作为笔记标题。例如本文，就是第一行的 欢迎使用 Markdown 编辑器。

离线存储 🗄

使用浏览器离线存储将内容实时保存在本地，不必担心网络断掉或浏览器崩溃。
